These are the CosmicFish Python applications with their documentation.

compare_cls_cov Simple Python code to plot a set of cls covariance.
compare_SN_mocks Simple Python code to plot a set of SN mocks.
plot_1D Simple Python code to do 1D plot of a set of Fisher matrices.
plot_2D Simple Python code to do 2D plot of a set of Fisher matrices.
plot_tri Simple Python code to do triangular plot of a set of Fisher matrices.
bounds Simple Python code to compute the narginal bounds of a set of Fisher matrices.
full_analysis Simple Python code to perform analysis of Fisher matrices (plot_1D, plot_2D, bounds...)
